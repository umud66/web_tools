Excel 同名工作簿批量合并

直接在浏览器中处理，不上传服务器。选择多个 Excel 文件后，系统会按工作簿名称自动归类合并。 你可以为每个工作簿设置“前 N 行是公共标题”，合并时只保留第一份标题，其余文件仅追加标题之后的数据。

1. 选择文件

支持拖拽或点击选择多个 `.xlsx` / `.xls` 文件。所有文件应保持相同结构。

尚未选择文件

2. 设置每个工作簿的标题行数

例如工作簿 A 填 `2`，表示前两行是公共标题，保留一次；从第 3 行开始才参与跨文件拼接。

读取文件后会在这里列出所有检测到的工作簿。

3. 合并并导出

0 已选择文件数
0 检测到的工作簿数

规则说明

  • 同名工作簿会被合并到结果文件中的同一个工作簿里。
  • 标题行只保留第一份，后续文件会跳过对应的标题行数。
  • 如果某个文件缺少某个工作簿，该文件会自动跳过该工作簿。
  • 所有处理都在当前浏览器中完成，刷新页面前可重复调整参数重新导出。

合并结果预览
